# 文字转语音项目

这是一个基于 [Kokoro TTS](https://huggingface.co/hexgrad/Kokoro-82M) 的文字转语音 Web 应用，支持将输入的文字转换为自然的语音。


## 技术栈

### 后端
- Python
- Kokoro TTS


### 前端
- React



## 安装和运行

### 后端设置

1. 安装 Python 依赖：

> cd backend

> pip install -r requirements.txt
2. 运行后端服务：

> python3 app.py 

### 前端设置

1. 安装依赖：

> cd frontend-react

> npm install
2. 运行前端页面：

> npm run dev
